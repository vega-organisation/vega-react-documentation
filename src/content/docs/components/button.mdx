The `Button` component renders an accessible `<button>` with Vega design tokens, variant styles, and size presets.

## Preview

<ButtonDemo client:load />

## Installation

```bash
npm install vega-react-components
```

Import the package stylesheet once in your application root:

```tsx
import 'vega-react-components/style.css';

```

## Import

```tsx
import { Button } from 'vega-react-components';

```

## API

| Prop Name   | Type                                                | Default Value | Required | Description                                  |
| ----------- | --------------------------------------------------- | ------------- | -------- | -------------------------------------------- |
| `children`  | `ReactNode`                                         | —             | Yes      | Button label or nested content.              |
| `variant`   | `'primary' \| 'secondary' \| 'danger' \| 'success'` | `'primary'`   | No       | Visual style of the button.                  |
| `size`      | `'small' \| 'medium' \| 'large'`                    | `'medium'`    | No       | Padding and font scale.                      |
| `fullWidth` | `boolean`                                           | `false`       | No       | Stretch the button to 100% of its container. |

`Button` extends `ButtonHTMLAttributes<HTMLButtonElement>`, so native attributes (`onClick`, `disabled`, `type`, `aria-*`, `className`, etc.) are supported.
